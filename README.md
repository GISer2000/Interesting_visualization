# 可视化示例集

- 本项目持续更新一系列的可视化示例，所有示例均基于 Python 生态（如 Matplotlib、Seaborn、Geopandas）实现，适用于数据分析和地理信息科学（GIS）相关领域。

- 前面几个示例做了函数封装，但感觉封装之后不太好做自定义调整，索性后面的示例就按照边填代码边可视化的形式出图。
